# LoveTravel Child Theme

## WordPress Child Theme
Professional WordPress child theme extending the LoveTravel theme with custom enhancements for tribetravel.eu.

## Features
- Payload CMS integration
- Elementor templates and tools
- Adventure management system
- Admin utilities and tools

## Installation
1. Upload to `/wp-content/themes/lovetravel-child/`
2. Activate via WordPress admin
3. Configure permalinks (Settings → Permalinks → Save)
